# **App Name**: ElectroFix Landing

## Core Features:

- Centralized Content Display: Display all content, including the title, video, logo, and footer, in a centralized layout that is responsive across desktop and mobile devices.
- Embedded Video Player: Embed a 16:9 aspect ratio video player using an iframe, ensuring it is responsive and maintains its proportions across different screen sizes.
- WhatsApp Button: Implement a circular WhatsApp button with a customizable link for direct user interaction.
- Informational Footer Links: Include 'Privacy Policy' and 'Terms of Use' links in the footer, providing users with easy access to important legal information.

## Style Guidelines:

- Primary background color: Very dark navy blue (#03122C) to provide a sleek, professional base.
- Text color: White (#FFFFFF) for primary text to ensure high contrast and readability against the dark background.
- Secondary text color: Light gray (#C6C6C6) for secondary text elements, offering a softer contrast.
- WhatsApp button color: Green (#25D366) for the WhatsApp button, aligning with the app's brand color.
- Font: 'Inter', a modern sans-serif font for a clean and contemporary look throughout the landing page. Note: currently only Google Fonts are supported.
- Centralized layout to ensure a clean, focused user experience on both desktop and mobile devices.
- White speaker icon above the title to highlight the audio feature.
